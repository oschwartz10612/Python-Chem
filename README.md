# Python-Chem

Simple Python scripts to solve basic chemistry problems

## Other tools

- Deducing the ions in a polyatomic ionic compound from its empirical formula: https://www.wolframalpha.com/

### Possible results

`Mn2(CO3)3 = Mn3+ CO32-`

`VF2 = V2+ F-`

`NH4I = NH4+ I-`

`Co3(PO4)2 = Co2+ PO43-`
